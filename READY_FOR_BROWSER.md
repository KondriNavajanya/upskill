## 🎓 Example Flows

### Flow 1: Take an MCQ Test
```
1. Open: http://localhost:5173
2. Sign Up / Login
3. Click "Learning Lab" (Sidebar)
4. Select "DSA" → "Easy"
5. Click "Start Test"
6. Answer 10 questions (10 minutes)
7. View Results
8. Click "Get Explanation" to learn
```

### Flow 2: Solve a Coding Problem
```
1. Click "Coding Lab" (Sidebar)
2. Select "DSA" → "Easy"
3. Click "Solve" on any problem
4. Code editor opens
5. Select language (Python, JS, etc)
6. Write your solution
7. Click "Submit"
8. View score & test results
9. Click "Explanation" to understand
```

### Flow 3: View Performance Analytics
```
1. Go to "Results" (Sidebar)
2. See all your test scores
3. View statistics by topic
4. Track improvement over time
5. Analyze weak areas
```
